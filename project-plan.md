# 경기도 1월 여행 가이드 웹앱 프로젝트 계획서

## 📋 프로젝트 개요
Plan - Generator - Healer 방식으로 개발하는 경기도 여행 추천 웹 애플리케이션

### 목표
- 4가지 여행 타입별로 각 5개씩 최적의 코스 추천 (총 20개 코스)
- 실시간 날씨 정보를 반영한 코스 업데이트
- 지도 기반 경로 안내 및 여행 경비 계산
- 맛집 추천 및 숙박 예약 연동

---

## 🎯 여행 타입별 코스 계획

### 1️⃣ 타입 1: 퇴근 후 불멍 당일치기 (친구 2-3명)
**목표**: 퇴근 후 늦은 시간에도 갈 수 있는 불멍 & 담소 코스

#### 추천 코스 5개
1. **가평 무지개 독채펜션&글램핑**
   - 위치: 경기도 가평군
   - 소요시간: 서울에서 1시간 20분
   - 특징: 계곡 근처, 바베큐+모닥불, 평일 교통 원활
   - 예상 비용: 1인당 30,000-50,000원 (입장+식비)

2. **가평 글램탐다오**
   - 위치: 경기도 가평군
   - 소요시간: 서울에서 1시간 30분
   - 특징: 이국적 분위기, 독립 BBQ시설, 불멍 화로 제공
   - 예상 비용: 1인당 70,000-120,000원 (당일 이용)

3. **포천 마루 글램핑 카라반**
   - 위치: 경기도 포천시
   - 소요시간: 서울에서 1시간
   - 특징: 숯불/장작 제공, 초보자 친화적
   - 예상 비용: 1인당 35,000-50,000원

4. **화성 글램비 글램핑**
   - 위치: 경기도 화성시
   - 소요시간: 서울에서 1시간 10분
   - 특징: 고급형 텐트, 불멍+바비큐+미온수 수영장
   - 예상 비용: 1인당 80,000-100,000원

5. **평택 진위천 유원지 쉼터 (무료)**
   - 위치: 경기도 평택시
   - 소요시간: 서울에서 1시간 20분
   - 특징: 무료 캠프닉, 주차비 3,000원만 필요
   - 예상 비용: 1인당 10,000-20,000원 (식비만)

### 2️⃣ 타입 2: 어머니와 당일 온천욕 + 식사
**목표**: 부모님을 모시고 편안하게 온천과 맛있는 식사를 즐기는 코스

#### 추천 코스 5개
1. **이천 테르메덴 코스**
   - 온천: 이천 테르메덴 (독일식 온천, 규모 최대)
   - 식사: 이천 쌀밥정식 (이천쌀로 유명)
   - 소요시간: 서울에서 1시간
   - 예상 비용: 1인당 50,000-80,000원 (온천+식사)

2. **화성 율암온천 코스**
   - 온천: 화성 율암온천 (100% 천연 게르마늄 온천)
   - 식사: 화성 전통 한정식
   - 소요시간: 서울에서 40분
   - 예상 비용: 1인당 40,000-60,000원

3. **포천 신북온천 코스**
   - 온천: 포천 신북온천 스프링폴
   - 식사: 포천 이동갈비 + 산정호수 구경
   - 소요시간: 서울에서 1시간
   - 예상 비용: 1인당 50,000-70,000원

4. **석모도 미네랄온천 코스**
   - 온천: 강화 석모도 미네랄온천 (바다 전망 노천탕)
   - 식사: 강화도 새우젓정식 또는 순두부
   - 소요시간: 서울에서 1시간 30분
   - 예상 비용: 1인당 30,000-50,000원 (가성비 최고)

5. **양평 온천 + 두물머리 코스**
   - 온천: 양평 블룸비스타 스파
   - 식사: 양평 한우 또는 더덕구이
   - 추가: 두물머리 강변 산책
   - 소요시간: 서울에서 1시간
   - 예상 비용: 1인당 60,000-90,000원

4. **양평 블룸비스타 스파 1박2일**

5. **가평 온천 + 남이섬 1박2일**

   - 온천: 가평 온천 리조트
   - 관광: 남이섬, 자라섬, 아침고요수목원
   - 예상 비용: 1인당 180,000-280,000원

### 4️⃣ 타입 4: 혼자 떠나는 드라이브 여행
**목표**: 혼자 자유롭게 드라이브하며 사색과 힐링

#### 추천 코스 5개
1. **안산 시화방조제길 코스**
   - 경로: 오이도 → 시화방조제(11.2km) → 대부도
   - 명소: 달전망대(75m), 서해 일몰
   - 식사: 대부도 해산물 또는 바지락칼국수
   - 소요시간: 3-4시간 (서울 왕복 포함)
   - 예상 비용: 30,000-50,000원

2. **양평 두물머리 강변길 코스**
   - 경로: 양수리 → 두물머리 → 수입리 강변길
   - 명소: 두물머리 일몰, 세미원, 양평군립미술관
   - 식사: 양평 더덕정식 또는 메밀국수
   - 소요시간: 4-5시간
   - 예상 비용: 40,000-60,000원

3. **파주 자유로 코스**
   - 경로: 가양대교 → 자유로 → 임진각 자유의 다리
   - 명소: 오두산통일전망대, 헤이리 예술마을, 출판도시
   - 식사: 파주 장단콩 두부 또는 임진각 막국수
   - 소요시간: 5-6시간
   - 예상 비용: 50,000-70,000원

4. **화성 화옹방조제 코스**
   - 경로: 매향리 → 화옹방조제(10km) → 궁평항
   - 명소: 제부도 바닷길, 궁평항 일몰, 전곡항
   - 식사: 궁평항 대하구이 또는 전곡항 회센터
   - 소요시간: 4-5시간
   - 예상 비용: 50,000-80,000원

5. **남양주 광릉수목원로 코스**
   - 경로: 축석검문소 → 광릉수목원 → 국립수목원 → 왕숙천
   - 명소: 국립수목원, 봉선사, 정약용 유적지
   - 식사: 남양주 닭갈비 또는 다산 유기농 정식
   - 소요시간: 4-5시간
   - 예상 비용: 40,000-60,000원

## 🏗️ 기술 스택 및 아키텍처

### Frontend
- **프레임워크**: React + TypeScript
- **스타일링**: Tailwind CSS
- **지도**: Kakao Map API 또는 Google Maps API
- **상태관리**: React Context API / Zustand

### Backend
- **프레임워크**: Node.js + Express.js
- **데이터베이스**: MongoDB (여행 코스 정보 저장)
- **캐싱**: Redis (날씨 정보 캐싱)

### 외부 API
- **날씨 API**: OpenWeatherMap API (실시간 날씨 정보)
- **지도 API**: Kakao Map API (경로 안내)
- **숙박 예약**: 여기어때/야놀자 API 연동 (가능시)
- **맛집 정보**: 카카오 로컬 API / 네이버 지도 API

### Testing
- **E2E Testing**: Playwright
- **Unit Testing**: Jest
- **Component Testing**: React Testing Library

## 📱 주요 기능 명세

### 1. 홈 화면
- 4가지 여행 타입 선택 카드
- 현재 날씨 정보 표시
- 추천 인기 코스 미리보기

### 2. 여행 타입별 목록 화면
- 5개 코스 리스트 표시
- 각 코스별 썸네일, 제목, 간단한 설명
- 예상 비용 범위 표시
- 소요 시간 표시
- 날씨 정보 배지 (오늘 날씨에 적합한지)

### 3. 코스 상세 화면
- **지도 섹션**:
  - Kakao Map으로 경로 표시
  - 출발지-경유지-목적지 마커
  - 자차 기준 내비게이션 연동

- **여행 정보 섹션**:
  - 상세 일정 타임라인
  - 각 장소별 설명
  - 소요 시간 및 거리

- **비용 섹션**:
  - 항목별 예상 비용 (통행료, 주차비, 입장료, 식비 등)
  - 총 예상 비용 계산
  - 인원수 조절 가능

- **맛집 추천 섹션**:
  - 코스 근처 맛집 리스트
  - 리뷰 점수 및 대표 메뉴
  - 지도에서 위치 확인
  - 예약/전화 연결

- **숙박 정보 섹션** (1박2일 코스):
  - 추천 숙박 시설 리스트
  - 가격대 및 시설 정보
  - 예약 사이트 연결 (여기어때, 야놀자 등)

- **날씨 정보**:
  - 여행 당일 예상 날씨
  - 주의사항 및 준비물 추천

### 4. 날씨 기반 추천 시스템
- 실시간 날씨 API 연동
- 날씨에 따른 코스 추천 순위 조정
- 비 오는 날 → 실내 온천 코스 우선 추천
- 맑은 날 → 드라이브/야외 코스 우선 추천
- 눈 오는 날 → 온천+눈 경치 코스 추천

### 5. 필터링 및 검색
- 예산별 필터 (3만원 이하, 5만원 이하, 10만원 이하 등)
- 거리별 필터 (30km 이내, 50km 이내, 100km 이내)
- 테마별 검색 (온천, 드라이브, 캠핑, 맛집 등)

## 🧪 Testing 전략 (Plan - Generator - Healer)

### Phase 1: Plan (계획)
✅ **현재 단계**
- 프로젝트 요구사항 분석 완료
- 기술 스택 선정 완료
- 아키텍처 설계 완료
- 20개 여행 코스 데이터 수집 완료

### Phase 2: Generator (코드 생성)
**테스트 우선 개발 (TDD)**

1. **E2E 테스트 작성** (Playwright)
   - 사용자 시나리오별 테스트 케이스 작성
   - 예: 홈 → 타입 선택 → 코스 목록 → 상세 보기 → 지도 확인

2. **컴포넌트 테스트 작성**
   - React Testing Library로 각 컴포넌트 테스트
   - 예: TravelTypeCard, CourseCard, MapSection 등

3. **Unit 테스트 작성**
   - 유틸리티 함수 테스트
   - 예: costCalculator, dateFormatter 등

4. **실제 코드 구현**
   - 테스트를 통과하는 최소 기능 구현

## 📝 API 엔드포인트 설계

### Course API
- `GET /api/courses` - 모든 코스 조회
- `GET /api/courses/:id` - 특정 코스 상세 조회
- `GET /api/courses/type/:type` - 타입별 코스 조회
- `GET /api/courses/recommend` - 날씨 기반 추천 코스

### Weather API
- `GET /api/weather/current/:location` - 현재 날씨 조회
- `GET /api/weather/forecast/:location` - 날씨 예보 조회

### Restaurant API
- `GET /api/restaurants` - 모든 맛집 조회
- `GET /api/restaurants/:id` - 특정 맛집 조회
- `GET /api/restaurants/nearby` - 근처 맛집 조회 (위도/경도 파라미터)

### Accommodation API
- `GET /api/accommodations` - 모든 숙박 조회
- `GET /api/accommodations/:id` - 특정 숙박 조회
- `GET /api/accommodations/nearby` - 근처 숙박 조회
